# TA_Management_PBO

Program ini merupakan aplikasi Sistem Manajemen Judul Tugas Akhir berbasis Java
yang dikembangkan untuk memenuhi tugas UAS Pemrograman Berorientasi Objek (PBO).

Aplikasi menerapkan konsep-konsep utama Pemrograman Berorientasi Objek serta
terintegrasi dengan database MySQL menggunakan JDBC.

## Fitur Utama
- Menambahkan judul Tugas Akhir
- Menampilkan data judul Tugas Akhir
- Mengubah status judul Tugas Akhir
- Menghapus data judul Tugas Akhir

## Notes
Program dijalankan melalui console dan menggunakan database sebagai media penyimpanan data.
